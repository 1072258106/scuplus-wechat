# SCUPLUS - 微信小程序[We川大]

> 这是一个主要服务四川大学师生的一个微信小程序，包含最新资讯/成绩/课程表/考试信息/图书借阅信息等

小程序采用wepy开发，参考了We重邮的界面设计
